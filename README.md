# R-E-Dine
## Example Restaurant Preorder & Takeaway System

Built with Node.js v20.9.0 & Electron 27.1.0

Usage:

1. Clone

2. Install electron (node.js is needed)
```bash
cd rootfolder/
npm install --save-dev electron
```

3. Start
```bash
npm start
```
